== Description ==

Enables an interstitial pop-up message to users when clicking a link to an external URL. Requires Advanced Custom Fields.

== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/external-link-popups` directory, or install the plugin through the WordPress plugins screen directly.
1. Activate the plugin through the 'Plugins' screen in WordPress
